== Description ==

WPC Composite Products provide a powerful kit-building solution for WooCommerce store. Buyer can build product kits in one page before buying.

= Live demo =

Click to see [live demo](https://demo.wpclever.net/wooco/ "live demo")

= Features =

- AJAX search & pick categories/products
- Set the discount by percentage
- Set default product for each component
- Set the default quantity for each component product
- Set optional for each component product
- Custom quantity with min/max value
- Auto calculate the price based on component products
- WPML integration

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Composite Products", Install & Activate it
4. Now when you create/edit a product you can choose product data is "Composite product", then add many components as you want
5. Almost done! click Save and see the result
